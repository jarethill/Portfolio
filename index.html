Dota Draft

Dota Draft takes match data from the official Dota 2 API and generates statistics on the backend to show hero and item winrates in coorelation with the other heroes in the game. This data is then sent to the frontend via REST API to provide suggestions to make sure the player selects a favorable hero vs the enemy’s draft.

Sudoku Solver

A Sudoku solver built in React with Typescript. Uses Jest for Unit Testing. I took a different approach from traditional Sudoku solvers by using Object Oriented Programming to encapsulate the board, line, and cell logic into their respective classes. By doing it this way, each cell always keeps track via references of all the other cells in it's row/column/subgrid, which leads to much faster solve validation; since it doesn't have to manually iterate through each line on every cell. This approach also means the logic is completely seperated from the UI, making it easily portable to another frontend or even Vanilla Javascript.

Memory Game

A game to test your memory, built with React using Functional Components. Gain points by choosing on a picture you haven't clicked on yet. After each selection, the pictures randomize and you must select again. Selecting an already clicked image leads to gameover and your score resets. How long can your streak last?

pwgen

A secure password generator written in Python. Built as a CLI tool to allow easy use in scripts or other applications. By default it creates random length, alphanumeric passwords; however the defining feature is it’s ability to use the diceware algorithm to make robust, multi word passwords.
